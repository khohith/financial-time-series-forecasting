## 📌 Project Description

This project aims to predict stock prices using financial time series data by combining signal processing and deep learning techniques.

The time series data is transformed into a time–frequency representation using Short-Time Fourier Transform (STFT), and a Convolutional Neural Network (CNN) is used for prediction.

---

---

## 📊 Methodology

1. Data collection from Yahoo Finance
2. Data preprocessing and normalization
3. Conversion of time-series to spectrogram using STFT
4. CNN model training
5. Prediction of future stock prices
6. Performance evaluation using Mean Squared Error (MSE)
